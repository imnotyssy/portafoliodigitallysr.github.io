Proyecto académico

TESIS
NUTRIFAV

Sistema web para optimizar el control nutricional, el registro clínico y el seguimiento de pacientes.

Problema Procesos manuales, demoras y riesgo de errores en registros y cálculos.
Objetivo Digitalizar pacientes, historiales, IMC, planes y progreso nutricional.
Tecnologías HTML5, CSS3, PHP, base de datos local y gráficas de seguimiento.

RESUMEN

Lo más importante del proyecto de forma breve y visual.

Funciones clave

Registro de pacientes, historial clínico, planes alimenticios personalizados y consultas más ágiles.

Automatización

Cálculo de IMC e indicadores antropométricos para reducir errores y ahorrar tiempo operativo.

Valor del sistema

Mejora la organización, permite seguimiento visual del progreso y fortalece la atención nutricional.
